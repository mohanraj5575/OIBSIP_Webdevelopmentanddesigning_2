## Project Title
Personal Portfolio Website

## Description
This project is a personal portfolio website developed using HTML and CSS. It is designed to showcase personal information, educational background, technical skills, projects, and contact details in a clean and structured format. The website provides a simple and user-friendly interface for presenting professional details.

## Features
- Profile section with photo and personal information  
- About Me section describing interests and goals  
- Education details with academic performance  
- Technical skills and soft skills display  
- Projects section with brief descriptions  
- Extra curricular activities section  
- Contact information with email and phone links  
- Download resume button  
- Responsive design for different screen sizes  

## Technologies Used
- HTML
- CSS

## How to Run
1. Download or copy the project files  
2. Open the HTML file in any web browser  

## Output
The website displays a complete personal portfolio including profile details, education, skills, projects, and contact information in a visually organized layout.

## Conclusion
This project demonstrates the use of HTML and CSS to design a personal portfolio website. It helps in understanding layout structuring, styling, and presenting information effectively on a webpage.
